# 🚀 Space Shooter

**A 2D spaceship shooting game built with Qt & C++**

*Built as a 2nd Semester Object-Oriented Programming (OOP) course project at IIUC*

</div>

---

## 📖 About

Space Shooter is a classic 2D arcade-style game where you pilot a spaceship and blast enemy ships out of the sky. The project was developed to demonstrate core **Object-Oriented Programming** concepts — including classes, inheritance, encapsulation, and polymorphism — using the **Qt framework** with **C++**.

---

## 🎮 Gameplay

- Control your spaceship and navigate the battlefield
- Fire bullets at incoming enemy spaceships
- Avoid enemy fire and survive as long as possible
- Enemy ships move and attack in patterns

---

## ✨ Features

- 🛸 Player-controlled spaceship with smooth movement
- 💥 Bullet firing mechanics for both player and enemies
- 👾 Enemy spaceship AI with movement patterns
- 🎯 Collision detection between bullets and ships
- 🖥️ Built entirely with Qt's graphics scene and view system

---

| Technology | Purpose |

## 🚀 Getting Started

### Prerequisites

- [Qt Creator](https://www.qt.io/download) (Qt 5.x or later recommended)
- A C++ compiler (MinGW / MSVC / GCC)

### Installation & Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/ibrahim3761/QT-Project.git
   cd QT-Project
   ```

2. **Open in Qt Creator**
   - Launch Qt Creator
   - Go to `File → Open File or Project`
   - Select the `.pro` project file

3. **Build & Run**
   - Click the **Run** button (▶) or press `Ctrl + R`
   - The game window will launch

## 🎓 Academic Context

> This project was developed during **2nd Semester** of the B.Sc. in Computer Science & Engineering program at **International Islamic University Chittagong (IIUC)** as part of the **Object-Oriented Programming (OOP)** course.

The game demonstrates practical application of OOP concepts:
- **Classes & Objects** — Separate classes for Player, Enemy, Bullet, etc.
- **Inheritance** — Shared base class for game entities
- **Encapsulation** — Game state managed within objects
- **Event Handling** — Qt signals and slots for game events
